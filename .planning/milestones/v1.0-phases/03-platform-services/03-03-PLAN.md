    **SecretsPanel (claudeos-secrets/src/webview/secrets-panel.ts):**
    Singleton WebviewPanel manager (same pattern as HomePanel):

    - `static currentPanel: SecretsPanel | undefined`
    - `static createOrShow(context, client, treeProvider, onSecretChange?)` -- reveal or create. If `secretName` arg provided, after creation post `{command: 'selectSecret', name: secretName}` to webview.
    - Constructor: stores panel, client, treeProvider refs. Sets retainContextWhenHidden: true, enableScripts: true, localResourceRoots.
    - `_getHtmlForWebview(webview, extensionUri)` -- returns complete HTML string with CSP nonce. HTML structure:
      - Two-column layout: `.secrets-list` (left, ~300px fixed) and `.secret-detail` (right, flex-1)
      - Left panel: header with "Secrets" title and $(add) button. Scrollable list of secret items (name + category badge). Active item highlighted with accent border-left.
      - Right panel: either empty state ("Select a secret or create new") or form with fields:
        - Name (text input, readonly when editing existing)
        - Value (password input, with eye toggle button and copy button)
        - Category (text input with datalist of existing categories)
        - Tags (text input, comma-separated)
        - Save button (primary), Delete button (danger, only shown for existing), Cancel button (secondary)

    - Message handler (onDidReceiveMessage):
      - `getSecrets` -- client.listSecrets(), post {command: 'secretsList', data: secrets}
      - `getSecretValue` (name) -- client.getSecretValue(name), post {command: 'secretValue', data: {name, value}}
      - `saveSecret` ({name, value, category, tags, isNew}) -- if isNew: client.createSecret; else: client.updateSecret. Then if name === 'ANTHROPIC_API_KEY': client.setEnv('ANTHROPIC_API_KEY', value). Then refresh tree (client.listSecrets + treeProvider.update). Post {command: 'secretSaved'}. Call onSecretChange callback if provided.
      - `deleteSecret` (name) -- client.deleteSecret(name). Refresh tree. Post {command: 'secretDeleted'}. Call onSecretChange callback.
      - `copySecret` (name) -- client.getSecretValue(name), vscode.env.clipboard.writeText(value), post {command: 'copied'}
      - `confirmDelete` (name) -- vscode.window.showWarningMessage with modal confirmation. If confirmed, proceed with deleteSecret logic.
