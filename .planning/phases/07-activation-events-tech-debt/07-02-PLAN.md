Then implement in terminal-manager.ts:

1. Add `private exitedSessions = new Set<string>();` as class field.

2. Change notifySessionExit signature to accept sessionName:
```typescript
notifySessionExit(sessionId: string, sessionName?: string): void {
  if (this.exitedSessions.has(sessionId)) return;
  const entry = this.terminals.get(sessionId);
  if (entry) {
    this.exitedSessions.add(sessionId);
    entry.pty.onSessionExit();
    const name = sessionName ?? sessionId;
    vscode.window.showInformationMessage(`Session '${name}' has ended`);
  }
}
```

3. In closeTerminal(), add `this.exitedSessions.delete(sessionId);` before or after the existing cleanup.

4. In the handleTerminalClose callback (the onDidCloseTerminal handler), also call `this.exitedSessions.delete(sessionId);` when removing from the terminals map.

5. In dispose(), add `this.exitedSessions.clear();` after disposing all terminals.
