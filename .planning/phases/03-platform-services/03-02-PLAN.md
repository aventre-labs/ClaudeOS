    Class that persists shortcuts in `context.globalState` under key `claudeos.home.shortcuts`.
    - Constructor takes `vscode.ExtensionContext`
    - `getShortcuts(): Shortcut[]` -- reads from globalState, returns defaults if empty
    - `addShortcut(shortcut: Shortcut): void` -- appends and persists
    - `removeShortcut(id: string): void` -- filters out and persists
    - `reorderShortcuts(ids: string[]): void` -- reorders by given ID array and persists
    - Default shortcuts: New Session (claudeos.sessions.create, "add"), Open Home (claudeos.home.open, "home"), Refresh Sessions (claudeos.sessions.refresh, "refresh"), Open Secrets (claudeos.secrets.openEditor, "key"), Open Terminal (workbench.action.terminal.toggleTerminal, "terminal")

    **HomePanel (claudeos-home/src/webview/home-panel.ts):**
    Class managing the webview panel lifecycle:
    - `static currentPanel: HomePanel | undefined` -- singleton tracking
    - `static createOrShow(context, client, shortcutStore)` -- if currentPanel exists, reveal; else create new WebviewPanel with viewType "claudeos.home", title "ClaudeOS Home", column ViewColumn.One, options {enableScripts: true, retainContextWhenHidden: true, localResourceRoots: [extensionUri + 'webview']}
    - Constructor stores panel ref, sets up message handler, sets HTML via `_getHtmlForWebview()`
    - `_getHtmlForWebview(webview)` -- generates HTML string with CSP nonce. All HTML, CSS, and JS are embedded as template literals in this method (not loaded from separate files). Uses getNonce() for CSP. Structure:
      - CSS: Branded stylesheet with `--claudeos-accent: #c084fc` (purple-400), `--claudeos-gradient-start: #7c3aed`, `--claudeos-gradient-end: #c084fc`. Hero section gradient, `.sessions-grid` with `repeat(auto-fill, minmax(280px, 1fr))`, `.session-card` with hover accent border, `.shortcuts-grid` with `repeat(auto-fill, minmax(140px, 1fr))`, `.banner.warning` for API key setup. All colors use `var(--vscode-*)` except custom accent and hero gradient.
      - HTML: Hero section with ClaudeOS wordmark (SVG text-based, not image) and new-session button. API key banner div (hidden by default). "Recent Sessions" section with empty `.sessions-grid`. "Shortcuts" section with empty `.shortcuts-grid`.
      - JS: `acquireVsCodeApi()` once, DOMContentLoaded posts getRecentSessions + getShortcuts. Message listener handles: recentSessions (render cards with name, status badge, relative time, workdir snippet; click opens session), shortcuts (render cards with codicon + label; click executes shortcut), anthropicKeyStatus (show/hide banner). Helpers: timeAgo(), statusColor(). Uses vscode.setState/getState for scroll persistence.
    - Message handler (onDidReceiveMessage):
      - `createSession` -- vscode.commands.executeCommand('claudeos.sessions.create')
      - `openSession` (sessionId) -- vscode.commands.executeCommand('claudeos.sessions.openTerminal', {id: message.sessionId})
      - `getRecentSessions` -- client.listSessions(), filter to non-archived, sort by createdAt desc, take first 8, postMessage back with {command: 'recentSessions', data: sessions}
      - `getShortcuts` -- shortcutStore.getShortcuts(), postMessage back
      - `addShortcut` / `removeShortcut` / `reorderShortcuts` -- delegate to store, post updated list back
      - `executeShortcut` (commandId, args?) -- vscode.commands.executeCommand(commandId, ...(args || []))
      - `openSecrets` -- vscode.commands.executeCommand('claudeos.secrets.openEditor', 'ANTHROPIC_API_KEY')
    - `onDidDispose` -- clear currentPanel reference
    - `dispose()` -- dispose panel
