           ```
             }
           }, 2000);
           ```

         - Return 200 immediately after credential write + wizardState.complete()
         - Fire off the setup server close + code-server start + health check + SSE broadcast as a background async operation (void promise with error logging)
         - This prevents the HTTP request from timing out during the ~30s health check
